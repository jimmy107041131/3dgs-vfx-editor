Quick Guide

3D Viewport

  • Look — Left-click drag
  • Pan — Right-click drag
  • Dolly — Scroll wheel
  • Move — W A S D (hover viewport)
  • Up / Down — E / Q
  • Sprint — Hold Shift

Node Graph

  • Add node — Right-click on graph
  • Connect — Drag from output → input
  • Delete — Select node, press Delete
  • Pan — Left-click drag on background
  • 1 + click — Quick add Float
  • 2 + click — Quick add Vec2
  • 3 + click — Quick add Vec3
  • 4 + click — Quick add Vec4

Pipeline

SplatSource → MapGet → [effects] → MapSet → Transform → Renderer

  • MapGet — Extract per-splat properties
  • MapSet — Combine modified properties
  • Transform — Position / Rotation / Scale
  • Presets — Double-click to expand subgraph
